# Eternal Rift
“O mundo foi dividido por fendas mágicas chamadas Rifts. Essas fendas conectam dimensões diferentes, como reinos celestiais, áreas sombrias, florestas, cavernas e locais antigos. O jogador precisa explorar, ficar mais forte e descobrir o que causou essas fendas.”

Eternal Rift é um RPG 2D de fantasia onde cada recurso coletado, cada batalha e cada construção ajudam o jogador a sobreviver e evoluir em um mundo cheio de fendas mágicas e mistérios.
