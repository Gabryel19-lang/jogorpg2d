# Eternal Rift - Bosses supremos por bioma + flechas em baús

## Novidades
- Adicionados 3 bosses muito fortes, um em cada bioma principal:
  - **Imperador das Dunas** no Deserto.
  - **Dragão Glacial Antigo** no bioma Congelado.
  - **Hidra Venenosa** no Pântano.

## Recompensas dos bosses supremos
Ao derrotar cada boss supremo, o jogador recebe:
- moedas extras;
- poções;
- flechas;
- XP alto;
- +1 mana máxima.

## Flechas em qualquer tipo de baú
Agora qualquer baú pode dar flechas quando for aberto pela primeira vez:
- baú raro;
- baú da dimensão;
- baús de bioma;
- baús celestiais;
- baús elementais;
- outros baús comuns, se existirem no mapa.

## Base usada
Essa versão mantém:
- esposas em estilo NPC bonito;
- inventário com ícones escondidos;
- itens usáveis;
- troca de arma pelo inventário;
- armas em pixel art melhoradas.
